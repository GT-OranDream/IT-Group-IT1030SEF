# IT-Group-IT1030SEF
GROUP work

1.	Environment: Any modern browser (Chrome, Edge, Firefox).
2.	How to run:
    Clone the GitHub repository and open LOST and FOUND 5.0.html directly.
   	Or deploy the code to a static hosting service for online access.



Instructions
Post an item: Select the item type → Fill in the name, description and contact information → Submit the post
View the list: Filter item types and select a sorting method → Browse all item information
Mark as found: For lost items, click the Found button and fill in a remark to mark it as found
View statistics: Check real-time statistics and smart match suggestions
Delete data: Click Delete to remove a single item; click Clear All Data to erase all records

Core Features
Elegant Launch & Navigation
Beautiful startup page with one-click access to the main interface
Three main menu entries: Post Item, Item List, Statistics
Smooth page transition animations for fluid interaction experience
Lost & Found Item Publishing
Supports two types: I lost an item / I found an item
Form includes item name, detailed description, contact information with character limits
Form validation for required fields with automatic empty-value prompts
One-click form reset for convenient operation
Full-Featured Item List Management
Category filtering: All items / Lost items only / Found items only
Multi-dimensional sorting: Newest first / Oldest first / Item name A-Z
Single deletion: Remove unwanted item records
Batch clear: One-click deletion of all local data
Status marking: Lost items can be marked as found with automatic status update
Smart Match Recommendations
System automatically matches lost and found items based on item name + description keywords
Calculates matching scores and generates high-reliability return suggestions
Deduplicated display to avoid repeated matches
Real-Time Data Statistics
Visual statistic cards: Number of lost items, found items, and total items
Real-time data updates to intuitively show platform usage
Premium Interactive Experience
Toast notifications: Real-time feedback for all actions (post, delete, reset)
Responsive design: Perfectly adapts to computers, tablets, mobile phones and other devices
Dynamic visual effects: Gradient backgrounds, breathing glow, hover animations, frosted glass cards
Local Data Persistence
Data storage based on browser LocalStorage
Data remains intact after page refresh or browser closure
